本期带来安卓开发入门教程，首先确保你有一个Android Studio，然后我们开始吧！

android studio是安卓开发工具,安装教程自行准备和下载

配置完成后我们打开，即可看到一个加号，点击加号我们就会得到一下界面

左侧是你要编写的设备，比如第一个默认选择的是平板和安卓手机，往下依次是，手表，电视，汽车，我们一般是给安卓手机开发，所以保持第一个选择，然后2在看右边，右边是官方的示例模板,用于学习参考，由于本期教学的是认识项目各个功能，所以选择一个示例进行学习参考认识，我们选择Bottom Navigation Views Activity，底部带按键的项目，并点击下一步，如图

然后就是设置自己的app数据了

Name 是指你的app在手机上的名称，也就是好比微信，他就需要在这个地方写微信，然后在手机上就会显示微信的app名称
Package name 这个是指软件的包名，每个app都会有包名，好比身份证，同时包名和手机上的软件包名不能重复，否则会导致冲突
Save location 这个是指你编写这个项目的储存位置，记住路径不能有中文否则打包会出现报错
Language 是你想要用什么语言进行软件编写，目前很火的编程语言是java和kotlin，所以这里选择kotlin,因为谷歌在尝试让这个新语言代替java，如果想要用java也许看个人喜欢
Minimum SDK 这个是指你想最低在安卓几上面运行，一般选择安卓8以上，因为很多手机都是安卓8以上，除非你想适配低端机
